User Manual

First, you can run player information system program in Start Menu or program icon in your desktop. You will see this screen.

 

 

Second, you will see Log In form. You must type user name and password and Click OK button.

 

 

 

 

 

Third, you will see Player Information form.

 

 

 

 

 

 

 

 

In Player Information Form             : click All Player check box in All Player Section. You will see all player information such as player id, player name, player DOB, etc… and you can click Next button in All Player Section you will see next player. And you can also click Previous button in All Player Section you will see previous player. And you can First & Last button in All Player Section you will see first player and last player.

 

 

 

 

 

 

 

 

You can search player by player id, player name and part of player name in Search Section.

 

 

 

 

 

 

 

 

You can also search player by previous searching in Previous Search Section. You can click Back and Next button in Pervious Search Section, you will see previous and next player in previous searching.

 

 

 

 

 

 

 

 

You can exit the program, click File menu and click Exit.
